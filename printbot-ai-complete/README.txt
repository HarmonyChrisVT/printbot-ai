==========================================
NEED HELP?
==========================================

1. Make sure you extract ALL files from the zip first
2. Don't move files around - keep the folder structure
3. Run START_HERE from the main folder (not from inside app/)
4. If you get "permission denied" on Mac/Linux, run:
   chmod +x START_HERE.sh
   Then try again
